# Tower Project

Front end for [tower backend](https://github.com/EJL85/ejl-tower-server). My first fully functioning fully CRUDing full-stack webapp.

### Installing

Run npm install

```
npm install
```

Run react!

```
npm run start
```

## Built With

* [React](https://reactjs.org/) - Front-end Framework
